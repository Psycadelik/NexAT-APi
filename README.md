Here's the basic directory structure;
```
├── app

│   ├── __init__.py

│   ├── resources

│   ├── templates
│   │   └── index.html
│   ├── utils
│   │   └── __init__.py
│   └── views
│       └── __init__.py
├── README.md
├── requirements.txt
├── test
│   └── __init__.py
└── wsgi.py

```
